GO Twitter Trends
=================

In order to make this work add a `config.json` file with the format
```
{
    "APIKey": {{ your_API_key }},
    "APISecret": {{ your_API_secret_key }},
    "AccessToken": {{ your_access_token }},
    "AccessSecret": {{ your_access_secret_token }},
    "WOEID": {{ WOEID_for_the_desired_location }}
}
```
